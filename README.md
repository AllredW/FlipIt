## About The Project

FlipIt is a study aid for anyone who enjoys learning. It allows users to create, edit, and delete decks; as well as create, edit, and delete flashcards.

## Getting Started

Once completely built, this application will function as a deployed app on Heroku

### Local Installation / Testing

1. Clone the rep

```
git clone https://github.com/404pandas/PARSR.git
```

2. Install dependencies

```
npm run install
```

3. Seed the database

```
npm run seed
```

4. Launch the app in development environments

```
npm run develop
```

5. Visit:

[localhost]()

